3D Music House

Class Project
Timeline: Spring 2022
Tools: OpenGL, C++, XCode

3D Music House* is an OpenGL music player program with animation and accepts keyboard inputs.

This work is an individual final project for the course Computer Graphics, a course I took in Spring, 2022. The class covered the basics of graphics programming and OpenGL, including topics such as shaders, transformation, texture, lighting, keyboard inputs, and model loading. In addition to applying the techniques taught in class, I further explored the use of audio libraries and Blender program to enhance the quality of the project.

The project was carried out when Harry Styles just released his new album “You are Home”. As I listened to the music, I imagined myself sitting on the couch inside the room of the album art. The project started from this.

The finished program allows users to get close to (or go inside) the room using 'Enter' key. 'A' key starts animation. It also pauses and resets animation when the animation is finished. 'U' key changes the view upside down. 'R' key returns the user to the initial camera state. Mouse dragging is used to change your view, and scrolling is used to zoom in/out when the user is inside the room.

FMOD library was added to the program to make this program a "music player". Space bar is used to pause and resume the music. Arrow keys (up and down) are used to increase or decrease the volume. When developing the program, royalty-free music was used instead of the actual track from Harry Styles' album. In the future, I wish to further explore the possibilities of this project. Collaboration with musicians or artists would be a dream come true!
